Firebase Configuration

To enable uploads, configure your Firebase credentials in js/firebase-config.js

Setup Instructions:
  1. Go to Firebase Console
  2. Create a new project or select existing one
  3. Enable Authentication (Email/Password)
  4. Create a Firestore Database
  5. Enable Storage
  6. Copy your config to js/firebase-config.js
  7. Add admin email(s) to the allowed list
